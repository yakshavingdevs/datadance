# Contributing

Development of Datadance happens in the open on GitHub. We welcome contributions from the community.

Please read our [Contributing Guide](https://github.com/yakshavingdevs/datadance/blob/main/CONTRIBUTING.md) and [Code of Conduct](https://github.com/yakshavingdevs/datadance/blob/main/CODE_OF_CONDUCT.md) before getting started.

## Development

```bash
# Clone the repository
git clone https://github.com/yakshavingdevs/datadance.git
cd datadance

# Start the development server
deno task start
```

## Reporting issues

File bug reports and feature requests at [https://github.com/yakshavingdevs/datadance/issues](https://github.com/yakshavingdevs/datadance/issues).
